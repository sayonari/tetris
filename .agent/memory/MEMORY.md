# MEMORY

## プロジェクト概要
- オートプレイ観賞型ネオン3Dテトリス。公開URL: https://sayonari.github.io/tetris/
- Vite + TypeScript + Three.js。mainへのpushでGitHub Actionsが自動デプロイ
- 詳細な技術知見は `.spec/KNOWLEDGE.md` を参照（重複記載しない）

## 学習した知識・教訓
- ユーザーはビジュアル・演出の派手さを重視する（「ビカビカドカンと」）。控えめより過剰気味が正解
- ユーザーは音量・レイアウトなど体感のフィードバックを随時くれる。実装後は必ずブラウザ実機確認してから報告する
- AIの挙動変更は必ずヘッドレスシミュレーション（scratchpadのsim.ts方式：esbuildバンドル→node実行）で定量検証してから採用する
- 評価関数チューニングでは「幾何判定の見せかけ検出」に注意。実際の到達可能性（BFS）で裏取りする
- 検証用スクリプトはプロジェクト外（scratchpad）に置き、リポジトリを汚さない

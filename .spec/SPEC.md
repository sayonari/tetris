# SPEC - 技術仕様・要件定義

## 機能要件

### ゲームコア（ガイドライン準拠）
- フィールド：10×20（＋上部非表示バッファ4段）
- テトリミノ：標準7種、標準配色（I=シアン, O=黄, T=紫, S=緑, Z=赤, J=青, L=オレンジ）
- 回転システム：SRS（スーパーローテーションシステム）、ウォールキック完全実装（JLSTZ用・I用テーブル）
- ネクスト：7バッグ方式、5個先まで表示
- ホールド：1回/ミノ、ホールド表示あり
- ゴーストピース表示
- 重力：ガイドライン曲線 `(0.8-(level-1)*0.007)^(level-1)` 秒/セル、10ライン毎にレベルアップ
- ロックディレイ：500ms、移動/回転でリセット（最大15回）
- ソフトドロップ（20倍速、+1点/セル）、ハードドロップ（+2点/セル）
- T-spin判定：3コーナールール＋フロントコーナーでMini判定、キック(1,-2)系でフル昇格
- スコアリング：ガイドライン準拠（Single100/Double300/Triple500/Tetris800、T-spin S/D/T=800/1200/1600、Mini、Back-to-Back×1.5、コンボ50×n×level、パーフェクトクリアボーナス、全て×level）

### オートプレイ（メイン機能）
- Dellacherie評価関数（設置高さ・消去セル・行/列遷移・穴・井戸）による自動プレイ
- BFS経路探索で回転・横移動・ソフトドロップ経由の「滑り込み（tuck）」も実行
- ホールド活用の自動判断
- 人間らしい操作速度（ボタン連打テンポ）で仮想ボタンを押す
- 高レベル時（重力が速い時）は滑り込み経路を自動制限（誤動作防止）
- ゲームオーバー後は5秒で自動リスタート（観賞モード継続）

### 手動プレイ
- キーボード：←→移動 / ↓ソフトドロップ / Space ハードドロップ / ↑・X 右回転 / Z 左回転 / C・Shift ホールド / Enter スタート・ポーズ / A オート切替 / M ミュート
- ゲームパッド：Gamepad API（十字キー、A/B回転、ショルダーでホールド、↑ハードドロップ）
- オートプレイ中に人間がゲーム操作をすると即マニュアルモードへ切替

### 画面上のコントローラ表示
- 画面下部に小さなゲームパッド（SVG）を常時表示
- 押されたボタンが発光（人間操作・AI操作どちらも同じ入力バスを通るため、オートプレイ時もAIの操作に合わせて光る）

### 演出・グラフィック（3D）
- Three.js によるWebGL 3D描画：角丸ブロック、ネオン発光フレーム、背景スターフィールド
- UnrealBloomPass によるブルーム（発光）ポストプロセス、ACESトーンマッピング
- ライン消去：フラッシュ＋パーティクル爆発＋画面シェイク、Tetris/T-spinは衝撃波リング＋強シェイク
- ハードドロップ：着地パーティクル＋小シェイク
- アクションテキスト（TETRIS! / T-SPIN DOUBLE! / B2B / COMBO / PERFECT CLEAR）のフローティング表示
- ピース移動は補間でヌルヌル動く

### サウンド
- Web Audio APIでリアルタイム合成（外部音源ファイル不使用＝著作権クリーン）
- BGM：コロブチカ（Korobeiniki、PD楽曲）のチップチューン風シーケンス、レベルでテンポアップ
- 効果音：移動/回転/ロック/ハードドロップ/ホールド/消去(段数別)/T-spin/レベルアップ/ゲームオーバー等
- 初回クリック（タイトル画面）でAudioContext起動（ブラウザ自動再生制限対応）

## 非機能要件
- GitHub Pages（https://sayonari.github.io/tetris/）で公開、GitHub Actionsで自動ビルド・デプロイ
- 外部アセット非依存（フォントのGoogle Fonts以外）でオフラインビルド可能
- 60fps動作目標、依存は three のみで軽量

## 技術構成
- Vite 6 + TypeScript（strict）
- Three.js（3D描画・ポストプロセス）
- Web Audio API（BGM/SFX合成）
- HTML/CSS オーバーレイ（HUD・コントローラSVG・オーバーレイ画面）
- デプロイ：GitHub Actions（actions/deploy-pages）、vite base './'

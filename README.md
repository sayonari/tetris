## 特徴

- **オートプレイ観賞モード**：AIが戦略を切り替えながら自動でプレイ
  - 🧠 **T-SPIN HUNT**：T-spinスロットを構築してTSD/TSTを狙う
  - 🧠 **REN COMBO**：左3列を温存して積み上げ、連続コンボで消す
  - 🧠 **TETRIS RUSH**：右端を温存して9列を整地し、I棒で4列消し
  - 危険な高さになると自動で立て直しモードに退避
- **プレイヤーの気持ち表示**：状況に応じたつぶやきを表示（90種以上）
- **ガイドライン準拠**：SRS回転・7バッグ・ホールド・ロックディレイ・T-spin判定・B2B・コンボ
- **3Dグラフィック**：Three.js＋Bloom、パーティクル・衝撃波・画面シェイクなどの派手な演出
- **画面上のコントローラ表示**：AI／人間の操作に合わせてボタンが光る
- **合成BGM/SFX**：Web Audioによるリアルタイム合成（コロブチカのチップチューンアレンジ）
- **手動プレイ対応**：キーボード＆ゲームパッド

## 開発

```bash

npm run build    # ビルド（tsc + vite）
npm run preview  # ビルド結果のプレビュー
```

（2026-07-16 プロジェクト初期化時にユーザーが記載した内容）

- このプロジェクトではテトリスのゲームを作りたい
- キーボードやコントローラを使ってプレイも可能だが、基本的には、システムのオートプレイを見て楽しむものにしたい
- テトリスの色や形は、標準のものに準拠すること
- テトリスの動作（回転の速さや動作の速さ、移動速度、接地した際の最終ロックまでの時間や、穴への滑り込み方、T-spinなど据えて）は、代表的なテトリスのルールに準拠させること
- 画面には小さいコントローラも表示され、押したボタンが表示されるようにしたい
  - オートプレイのときにもちゃんと動作に合わせてそのボタンが光るようにして欲しい
- その他、最新のゲーム構築環境と、最新のアセット、BGM、効果音、グラフィックなどをふんだんに使うこと
- 3D表示をさせても良いので、きれいなグラフィック、かっこいいグラフィック、かっこいい演出、かっこいい効果などふんだんに盛り込むこと
- 最終的には GitHub Pages（github.io）経由で公開し、ブラウザで遊べるようにしたい
